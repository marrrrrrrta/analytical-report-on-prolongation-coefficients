# Анализ пролонгаций договоров

Тестовое задание на позицию Data Analyst. Расчёт коэффициентов пролонгации договоров по менеджерам и месяцам за 2023 год с выгрузкой результатов в Excel.

---

## Описание задачи

Отдел сопровождения клиентов ведёт портфель договоров. Каждый договор имеет последний месяц активной отгрузки. Задача — определить, какая доля завершившихся договоров возобновляется (пролонгируется) в последующие месяцы.

Рассчитываются два коэффициента:

- **К1** — доля договоров, пролонгированных в 1-й месяц после завершения
- **К2** — доля договоров, пролонгированных во 2-й месяц (из тех, что не пролонгировались в 1-й)

Расчёт ведётся для каждого менеджера и по отделу в целом за каждый месяц 2023 года, а также в виде годового взвешенного коэффициента.

## Методология

### Коэффициент К1

```
Знаменатель = Σ отгрузка[M-1]  по проектам, завершившимся в M-1
Числитель   = Σ отгрузка[M]    по тем же проектам, у которых есть отгрузка в M

К1 = Числитель / Знаменатель
```

### Коэффициент К2

```
База        = проекты из M-2, не имеющие отгрузки в M-1
Знаменатель = Σ отгрузка[M-2]  по проектам базы
Числитель   = Σ отгрузка[M]    по проектам базы, у которых есть отгрузка в M

К2 = Числитель / Знаменатель
```

### Годовой коэффициент

Считается как взвешенное среднее: сумма всех числителей за 2023 делится на сумму всех знаменателей. Это корректнее, чем среднее арифметическое месячных значений.

### Правила обработки данных

| Значение | Обработка |
|----------|-----------|
| `стоп` / `end` | Проект исключается из расчёта полностью |
| `в ноль` | Берётся отгрузка ближайшего предыдущего ненулевого месяца |
| Дубли строк (один `id`) | Числовые значения суммируются; `стоп`/`end` приоритетнее чисел |

АМ (аккаунт-менеджер) берётся из `prolongations.csv` — эти данные первичны по отношению к `financial_data.csv`.

## Запуск

### В Google Colab (рекомендуется)

1. Открыть [colab.research.google.com](https://colab.research.google.com) → **Файл → Загрузить ноутбук** → выбрать `prolongation_analysis.ipynb`

4. После выполнения последней ячейки файл `prolongation_report.xlsx` скачается автоматически
